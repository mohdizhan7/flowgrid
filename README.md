# Flowgrid v1.5 (No-DB demo)

This is a ready-to-run Next.js + Tailwind build of Flowgrid with in-memory data (no database).

## Run locally
```bash
npm i
npm run dev
# open http://localhost:3000
```

## Deploy (Vercel)
- Push this folder to GitHub, then import into Vercel.
- No env vars needed for this demo.
